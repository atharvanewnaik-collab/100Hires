# Transcript Collection Script

This folder contains a sample Python script demonstrating how YouTube transcripts can be collected programmatically using the `youtube-transcript-api` library.

## Dependency

```bash

pip install youtube-transcript-api

```

## Example Usage

```bash

python fetch_[transcript.py](http://transcript.py)

```

The script accepts a YouTube video ID, retrieves the transcript, and prints the transcript text. This demonstrates the API-based workflow used for transcript collection before organizing and summarizing the research material.
